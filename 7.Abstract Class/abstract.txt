#Customizing Field Names:
You can use the source attribute to specify a different attribute or method name 
from the model to be used when serializing or deserializing a field

#Accessing Nested Attributes:
If you have nested attributes or properties in your model, you can use source to 
navigate to those attributes and serialize them accordingly.
